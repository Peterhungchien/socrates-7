# Curriculum Map

Maps each curriculum section to its source textbook and page range.
Used by hybrid NLM mode to load the correct section PDF at each topic transition.

**To populate:** During initialization (step 8 of `docs/init_prompt.md`), the teacher will auto-populate this table using NLM by querying your textbook's structure. Review the generated map, correct any page numbers, then run:
```bash
pip install pypdf
python tools/split_pdf.py
```
